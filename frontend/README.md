# BookIt Frontend (Next.js + TypeScript + Tailwind)

## Setup
1. Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_API_URL` to your backend URL (default: http://localhost:5000)
2. Install dependencies:
   ```
   npm install
   ```
3. Run dev:
   ```
   npm run dev
   ```
Open http://localhost:3000
